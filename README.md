Trabajo final de Programacion orientada a objetos I, 

El proyecto consiste en un sistema de gestión integral para ServiFabrik, una mini empresa dedicada a la confección, bordados y serigrafía.
El sistema permite administrar los clientes, empleados, productos, inventario y proveedores, así como gestionar todo el proceso de venta desde que el cliente realiza un pedido hasta que se genera la factura y se registra el pago.

El flujo principal inicia cuando un cliente realiza un pedido, el cual es atendido por un empleado. Cada pedido contiene uno o varios detalles que especifican los productos solicitados. 

Los productos pertenecen a una categoría y su disponibilidad se controla mediante el inventario, abastecido por proveedores. Una vez confirmado el pedido, se genera una factura y posteriormente se registran los pagos correspondientes.

El objetivo del sistema es mejorar el control de pedidos, evitar errores en entregas, mantener actualizado el inventario y automatizar procesos clave para la operación diaria de la empresa. Estaremos utilizando los conocimientos previos de la clase
tales como las notaciones de entidades, conexion a bases de datos, y por su puesto utilizaremos el framework OpenXava, y ocuparemos el IDE o editor de codigo Intellijidea
